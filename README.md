## 🌟 Key Features
- 📊 **Mood Calendar**: Full monthly visual grid mapping daily emotional trends and study stressors.
- 🧘‍♂️ **Relax Center**: Dynamic Pomodoro timer and an interactive, physics-based canvas Zen Bubble Popper game.
- 🚨 **Crisis Bypass**: Instant interceptor that bypasses LLMs to display helpline cards for suicidal/crisis keywords.
- 🛡️ **PII Scrubbing**: Client-side regex protection to scrub emails, phone numbers, and names before LLM requests.
- 💬 **Empathetic Companion**: Conversational bot grounded in WHO stress-management guidelines (rejects homework/coding requests).
- 🎨 **Teal/Mint Theme**: Sleek glassmorphic theme with a zero-flicker light/dark toggle.
- 📦 **Local JSON Cache**: Zero-downtime offline fallback database matching Vercel schema requirements.

---

## 🛠️ Tech Stack
- **Next.js 15 (App Router)**: Fast Server-Side Rendering (SSR) & serverless edge routes.
- **Supabase (PostgreSQL)**: Secure database with row-level security (RLS) policies.
- **Groq (Llama 3.3 70B)**: Real-time mental wellness trigger extraction (<1s latency).
- **Tailwind CSS v4**: Modern styling with customizable HSL variables.
